#  Professional Financial Management API

This is a high-level backend application for financial management, built with a robust architecture using Node.js, TypeScript, and PostgreSQL.

##  Technologies
- **Node.js** & **Express** (Server)
- **TypeScript** (Strong typing)
- **Prisma ORM** (Database management)
- **PostgreSQL** (Database)

##  How to run the project
1. Clone the repository
2. Install dependencies: `npm install`
3. Configure your `.env` file with your database URL
4. Run migrations: `npx prisma migrate dev`
5. Start the server: `npm run dev`

#  API de Gestão Financeira Profissional

Esta é uma aplicação backend de alto nível para gestão financeira, construída com uma arquitetura robusta utilizando Node.js, TypeScript e PostgreSQL.

##  Tecnologias
- **Node.js** & **Express** (Servidor)
- **TypeScript** (Tipagem forte)
- **Prisma ORM** (Gerenciamento de banco de dados)
- **PostgreSQL** (Banco de dados)

##  Como rodar o projeto
1. Clone o repositório
2. Instale as dependências: `npm install`
3. Configure seu arquivo `.env` com a URL do seu banco de dados
4. Rode as migrations: `npx prisma migrate dev`
5. Inicie o servidor: `npm run dev`
